Bài 6: Liệt kê các phương pháp clone object (càng nhiều càng tốt), phân biệt shallow clone và deep clone ? Có phải trường hợp nào cũng nên deep clone không ? Vì sao.
Trả lời: 
- Phương pháp clone object:
+ Sử dụng dấu ba chấm ... const obj2 = {...obj1} (shallow clone) 
+ Sử dụng Object.assign() const obj2 = Object.assign({}, obj1) (shallow clone)
+ Sử dụng JSON.stringify() để bọc object cần clone sau đó sử dụng JSON.parse() để parse 1 json string qua json. const obj2 = JSON.parse(JSON.stringify(obj1)) (deep clone).
- Phân biệt:
+ Với shallow clone thì phương pháp này chỉ clon được những object ở lớp bên ngoài, còn những con bên trong object đó thì không clone được. Dẫn đến bị mất mát dữ liệu. (Đã gặp trong TH thực tế)
+ Với deep clone thì phương pháp này sẽ clone toàn bộ object từ lớp ngoài tới lớp trong.
- Không phải lúc nào cũng nên sử dụng deep clone. Deep clone chỉ nên sử dụng khi cần clone toàn bộ object con ở bên trong 1 object còn những trường hợp chỉ cần lớp object bên ngoài hoặc object chỉ 1 lớp bên ngoài thì nên sử dụng shallow clone.
- Vì deep clone sẽ chạy mất nhiều thời gian hơn khi thực thi code. Việc này nếu xảy ra thì sẽ gây ảnh hưởng tới performance.
